# Task Manager Web Application



A simple Task Manager built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Display task list
- Mark tasks as completed
- Delete tasks
- Responsive design
- Dynamic DOM manipulation
- No page reload required

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

Task-Manager/

- min2.html
- min2.css
- min2.js

## How to Run

1. Open the project folder in VS Code.
2. Install the Live Server extension (optional).
3. Open `min2.html`.
4. Click **Open with Live Server**, or simply open `min2.html` in your browser.

## Functionalities

- Click **Add Task** or press **Enter** to add a task.
- Click **✔** to mark a task as completed.
- Click **✖** to delete a task.
