# Password Generator Web Application

## Objective

A Password Generator built using HTML, CSS, and JavaScript.

## Features

- Generate secure random passwords
- Adjustable password length (4–30)
- Include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Copy password to clipboard
- Input validation
- Responsive design

## Technologies

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download the project.
2. Open the folder in Visual Studio Code.
3. Open `min.html`.
4. Right-click → Open with Live Server
   or simply open `min.html` in your browser.
